# FDF

## Project Overview

This project is about creating a simplified 3d view of a relief map

## Installation

```
$> git clone https://github.com/bmoiroud/fdf.git && cd fdf && make
```

## usage

```
$> ./fdf <map>
```

| Key         | Action         |
|:-----------:|:---------------|
| +/-         | zoom in/out    |
| w/s         | z+/z-          |
| R           | reset view     |
| F           | fill           |
| Esc         | exit           |
| Arrows      | move around    |
